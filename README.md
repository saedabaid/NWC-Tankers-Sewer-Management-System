# NWC Tankers & Sewer Management System

## Overview

An internal enterprise web application developed for the **National Water Company (NWC)** to support and streamline tanker and sewer service operations.

The system manages operational workflows, service requests, scheduling, and tracking processes while integrating with **Elm (ELM) APIs** to enable secure data exchange and automated business operations.

## Key Features

* Service request management
* Tanker operations tracking
* Sewer service workflow management
* Scheduling and task assignment
* Status tracking and operational monitoring
* User roles and permissions management
* Background job processing and automation
* Integration with external services through ELM APIs
* Reports and operational insights
